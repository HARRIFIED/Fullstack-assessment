# User Account Management System

This is a user management system api that creates and manages user account.
A user can create account, fund account, transfer to another account, withdraw,
and also check available balance

## Features

- **Create Account**
- **Fund Account**
- **Transfer Funds**
- **Withdraw Funds**

### Bonus Feature: Added robust authentication

## Installation and Setup

Follow the steps below to get the project up and running on your local machine.

### Prerequisites

- **Node.js** (v16+)
- **npm** or **yarn**

### Installation

1. **Clone the repository**:

   ```bash
   git clone https://github.com/HARRIFIED/Fullstack-assessment.git
   cd backend_test/

2. **Download and install mongoDB on your machine**
   https://www.mongodb.com/try/download/shell
3. **or [OPTIONAL] use mongodb cloud and obtain url then modify connection code**

4. **Create database and connect using MongoDB shell or MongoDB compass**
   
5. **Create .env file**
   ````
       //example env file
       DB_HOST=localhost
       DB_PORT=27017
       DB_NAME=account_management
       JWT_SECRET=hello_word
       PORT=8080

6. **Install dependency**:

    ```bash
       yarn install
       npm install

7. **Run Application**:

     ```bash
       yarn dev
       npm run dev
   
8. **Testing**:

     ```bash
       yarn test
       npm run test
